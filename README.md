# The Swift Programming Language\(日本語版\)

[The Swift Programming Language](https://docs.swift.org/swift-book/)の日本語版です。

## バージョン履歴

- 2022/3/16 Swift5.6
- 2021/9/21 Swift5.5

## URL

[https://www.swiftlangjp.com](https://www.swiftlangjp.com)
